# 提案：使用 xlsx 技能分析現有數據進行股票回測

## 變更 ID
`xlsx-stock-analysis`

## 提案概述

基於項目現有的豐富股票數據資源，整合 xlsx 技能創建一個綜合性的股票分析和回測報告系統。該系統將利用 xlsx 技能強大的數據處理、可視化和分析能力，整合現有的股票技術指標數據、相關性分析和交易策略結果，生成專業級的量化分析報告。

## 背景與動機

### 現有資源
1. **股票數據集**：包含多隻香港股票（0001.HK、0005.HK等）的歷史數據
2. **技術指標結果**：6種技術策略的回測結果（MA、RSI、MACD、KDJ、BOLL、ADX）
3. **分析報告**：相關性矩陣、滯後相關性、預測力分析等
4. **投資組合數據**：現有的投資組合配置信息

### 挑戰與需求
- 現有數據分散在多個CSV文件中，缺乏統一的可視化分析
- 需要更直觀的方式比較不同策略的表現
- 需要整合多維度數據進行綜合分析
- 缺乏專業級的報告生成能力

## 解決方案

### 核心能力

#### 1. 數據整合與分析
- 使用 xlsx 技能讀取並整合所有現有CSV數據
- 創建統一的數據模型，包含股票價格、技術指標、策略表現
- 執行多維度數據分析（相關性、風險、回報）

#### 2. 策略比較與可視化
- 創建交互式圖表比較不同技術策略的表現
- 生成累積收益曲線、夏普比率、最大回撤等關鍵指標
- 實現多股票、多策略的並排比較

#### 3. 風險分析與組合優化
- 基於歷史數據計算投資組合的風險指標
- 生成風險-收益散點圖
- 提供策略配置建議

#### 4. 專業報告生成
- 自動生成包含圖表、分析和建議的完整Excel報告
- 包含執行摘要、詳細分析、風險評估等章節
- 支持導出PDF格式供分享
