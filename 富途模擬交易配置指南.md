# 富途模擬交易配置指南

## 🔍 重要發現

根據富途官方文檔檢查，發現了關鍵配置要求：

### ⚠️ 關鍵配置參數：`pdt_protection`

**問題根源**：OpenD 客戶端的防止數據盜用保護功能可能影響模擬交易API訪問。

**解決方案**：需要將 `pdt_protection` 參數設置為 0

## 🔧 具體解決步驟

### 方法一：FutuOpenD 命令行配置

1. **打開命令行**
2. **找到 FutuOpenD 安裝路徑**
3. **執行以下命令**：
   ```bash
   FutuOpenD.exe --pdt_protection=0
   ```

### 方法二：配置文件修改

1. **找到 OpenD 配置文件**
   - 路徑通常在：`C:\Users\{用戶名}\FutuOpenD\`
   - 配置文件：`FutuOpenDConfig.xml`

2. **修改配置**：
   ```xml
   <Setting name="pdt_protection" value="0" />
   ```

### 方法三：富途牛牛App設置

1. **打開富途牛牛App**
2. **進入設置**
3. **找到「開發者選項」或「API設置」**
4. **關閉數據保護選項**

## 🔍 基於錯誤分析的具體問題

**錯誤信息分析**：
- 錯誤代碼：-1
- 錯誤類型：字符串錯誤（包含IP地址）
- 可能原因：權限被 `pdt_protection` 阻止

## 🚀 解決後驗證步驟

### 步驟一：重啟服務
1. **完全關閉 FutuOpenD**
2. **按上述方法修改配置**
3. **重新啟動 FutuOpenD**

### 步驟二：驗證連接
```bash
python final_account_check.py
```
