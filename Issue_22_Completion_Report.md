## 使用指南

### 运行测试套件

```bash
# 1. 安装依赖
python setup_test_env.py

# 2. 启动API服务器
cd src/api && python -m uvicorn main:app --reload --port 3004

# 3. 运行测试套件
python test_api_suite.py
```

### 生产部署

```bash
# 1. 准备部署配置
python prepare_deployment.py

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件设置生产环境值

# 3. 构建Docker镜像
docker build -t cbsc-api:latest .

# 4. 测试容器运行
docker run -p 3004:3004 cbsc-api:latest

# 5. 生产部署
./deploy.sh
```
