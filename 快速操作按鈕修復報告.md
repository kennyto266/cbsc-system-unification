**修代碼示例：**
```typescript
// 處理創建新策略
const handleCreateStrategy = async () => {
  setLoading('create');
  try {
    router.push('/dashboard/strategies?action=create');
  } catch (error) {
    console.error('創建策略失敗:', error);
    alert('創建策略失敗，請稍後重試');
  } finally {
    setLoading(null);
  }
};
```

### 3. API 代理路由
為了解決 CORS 問題，創建了以下代理路由：

- **/api/data/stocks/route.ts** - 市場數據代理

- **/api/export/strategy/[id]/route.ts** - 導出報告代理

所有代理都包含：
- 轉發請求到後端 (端口 3004)
- 錯誤處理和降級方案
- 返回演示數據當後端不可用時

### 4. 環境配置 (next.config.ts)
