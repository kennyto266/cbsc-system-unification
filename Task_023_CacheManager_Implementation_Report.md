## 🚀 部署和使用

### 初始化
```python
from services.cache_manager import CacheManager

# 创建缓存管理器
cache_manager = CacheManager(
    redis_host="localhost",
    redis_port=6379,
    enable_redis=True,
    default_memory_size=1000
)
```

### 基础操作
```python
# 设置缓存
cache_manager.set("user", "user_123", {"name": "John", "age": 30})

# 获取缓存
user_data = cache_manager.get("user", "user_123")

# 删除缓存
cache_manager.delete("user", "user_123")

# 批量清理
deleted_count = cache_manager.clear_pattern("user", "temp_*")
```

### 策略使用
```python
# 获取策略
strategy = cache_manager.get_strategy("performance")

# 自定义策略
from services.cache_strategy import CacheStrategy, CacheLevel
custom_strategy = CacheStrategy(
    name="custom",
    ttl_seconds=600,
    cache_level=CacheLevel.L1_L2,
    max_memory_items=500
)
cache_manager.register_strategy(custom_strategy)
```
