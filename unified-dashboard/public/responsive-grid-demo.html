🚀 CBSC響應式網格系統

一個強大的響應式網格佈局系統，專為CBSC量化交易Dashboard設計。 支持拖拽、調整大小、持久化等高級功能，與477技術指標系統深度集成。

📱

響應式設計

自動適應不同屏幕尺寸，從大屏顯示器到移動設備都能完美展示。 支持5個響應式斷點：lg(12列)、md(10列)、sm(6列)、xs(4列)、xxs(2列)。

🎯

拖拽支持

組件可以自由拖拽移動位置，實時預覽效果。 支持多選、批量操作、對齊輔助等高級功能。

📐

調整大小

拖拽組件邊角即可調整大小，支持最小/最大尺寸限制。 智能網格對齊，確保佈局整潔美觀。

💾

持久化存儲

自動保存用戶的佈局設置到本地存儲。 支持佈局導入導出，方便備份和分享。

🎨

豐富組件

內置多種Dashboard組件：市場概覽、技術指標、策略表現、資產配置等。 支持自定義HTML、Markdown、iframe內容。

⚡

性能優化

基於React的高性能渲染，支持虛擬化和懶加載。 智能更新機制，避免不必要的重渲染。

📝 快速開始

只需幾行代碼即可創建響應式Dashboard：

import { ResponsiveGridProvider, ResponsiveGrid } from './ResponsiveGrid'

function Dashboard() {
  return (
    <ResponsiveGridProvider>
      <ResponsiveGrid
        editable={true}
        showToolbar={true}
        onLayoutChange={(layout) => console.log(layout)}
      />
    </ResponsiveGridProvider>
  )
}
