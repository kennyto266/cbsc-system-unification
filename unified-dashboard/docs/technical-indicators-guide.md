# 技术指标库功能指南

## 概述

技术指标库是 CBSC Dashboard 的核心功能之一，提供了 477 种专业技术指标的可视化和配置工具。用户可以浏览、搜索、配置和组合各种技术指标，构建个性化的交易策略。

## 功能特性

### 1. 指标库浏览
- **分类展示**: 指标按类别组织（趋势、动量、波动率、成交量等）
- **智能搜索**: 支持按名称、描述、标签搜索
- **收藏管理**: 收藏常用指标，快速访问
- **指标详情**: 查看指标的公式、参数、信号规则等详细信息

### 2. 动态参数配置
- **实时调整**: 动态修改指标参数，即时预览效果
- **参数验证**: 自动验证参数范围和约束
- **预设模板**: 提供常用的参数组合模板
- **配置保存**: 保存自定义配置供后续使用

### 3. 策略组合编辑器
- **拖拽式界面**: 直观的拖拽操作组合多个指标
- **逻辑配置**: 设置指标之间的组合逻辑
- **实时预览**: 即时查看组合效果
- **性能分析**: 分析组合策略的历史表现

### 4. 可视化设置
- **样式定制**: 自定义指标的颜色、线型、透明度
- **图表布局**: 灵活调整图表布局和显示方式
- **信号标记**: 清晰标记买卖信号点
- **多时间框架**: 支持不同时间框架的切换

## 使用指南

### 访问技术指标库

1. 登录 CBSC Dashboard
2. 在左侧导航栏点击"技术指标库"
3. 进入指标库主页面

### 浏览和搜索指标

1. **浏览分类**：
   - 点击顶部的分类标签（趋势、动量等）
   - 查看该类别下的所有指标

2. **搜索指标**：
   - 在搜索框输入指标名称或关键词
   - 使用高级筛选进行精确搜索

3. **查看详情**：
   - 点击指标卡片查看详细信息
   - 包括公式、参数说明、信号规则等

### 配置指标参数

1. 在指标详情页面，点击"配置参数"
2. 调整参数值：
   - 使用滑块或输入框修改数值
   - 查看参数说明和建议范围
3. 实时预览效果
4. 保存配置

### 创建指标组合

1. 选择需要的指标，点击"添加到配置"
2. 在配置面板中：
   - 调整每个指标的参数
   - 设置图表布局
   - 配置视觉样式
3. 保存组合配置
4. 应用到图表

### 查看性能分析

1. 在指标详情页面切换到"性能"标签
2. 选择分析的时间范围和交易品种
3. 查看统计数据：
   - 成功率
   - 盈亏比
   - 最大回撤
   - 夏普比率

## 指标类别说明

### 1. 趋势指标
用于识别市场趋势方向和强度
- ADX (Average Directional Index)
- Aroon Oscillator
- MACD (Moving Average Convergence Divergence)
- Parabolic SAR

### 2. 动量指标
衡量价格变化的速度和力度
- RSI (Relative Strength Index)
- Stochastic Oscillator
- Williams %R
- Momentum
- Rate of Change

### 3. 波动率指标
反映市场波动性和风险水平
- Bollinger Bands
- Average True Range (ATR)
- Standard Deviation
- Chaikin Volatility

### 4. 成交量指标
基于成交量分析市场行为
- On-Balance Volume (OBV)
- Volume Weighted Average Price (VWAP)
- Money Flow Index (MFI)
- Accumulation/Distribution Line

### 5. 移动平均
平滑价格数据，识别趋势
- SMA (Simple Moving Average)
- EMA (Exponential Moving Average)
- WMA (Weighted Moving Average)
- DEMA (Double Exponential Moving Average)

## 最佳实践

### 1. 指标选择原则
- **避免重复**: 不要使用过多相似的指标
- **互补搭配**: 结合不同类型的指标
- **考虑周期**: 选择适合交易时间框架的指标
- **理解原理**: 充分理解指标的工作原理

### 2. 参数优化建议
- **从默认值开始**: 使用推荐的默认参数
- **逐步调整**: 小幅调整并观察效果
- **历史验证**: 使用历史数据验证参数效果
- **定期更新**: 根据市场变化调整参数

### 3. 组合策略构建
- **逻辑清晰**: 明确每个指标的作用
- **权重分配**: 合理分配不同指标的权重
- **风险控制**: 设置止损和止盈规则
- **回测验证**: 进行充分的历史回测

## 常见问题

### Q: 如何找到适合的指标？
A: 首先了解您的交易风格和策略目标，然后浏览相关类别的指标，阅读其说明和应用场景。

### Q: 参数如何优化？
A: 使用策略回测功能，测试不同参数组合的历史表现，选择风险收益比最优的参数。

### Q: 可以创建自定义指标吗？
A: 是的，系统支持创建自定义指标。在指标库页面点击"创建自定义指标"按钮。

### Q: 如何保存和分享配置？
A: 在配置面板中保存您的配置，可以选择公开分享给其他用户或导出为文件。
