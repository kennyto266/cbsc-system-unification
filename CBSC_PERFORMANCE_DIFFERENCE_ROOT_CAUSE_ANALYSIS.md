### 原因三：计算方法错误和逻辑矛盾

#### 🚨 问题1：最大回撤计算错误

**MACD策略异常：**
- 总收益率：-192.33%
- 最大回撤：0.00%
- **逻辑矛盾**: 亏损192%但无回撤，计算错误

**正确逻辑：**
```
如果总收益为-192.33%，则:
  - 资金从100,000变为-92,338
  - 最大回撤应该至少为192.33%
  - 当前0%结果是明显的计算错误
```

#### 🚨 问题2：年化收益率计算错误

**MACD策略NaN值：**
- 年化收益率：NaN（不是数字）
- **原因**: 可能是负资金导致的数学错误

**MACD Standard vs Sensitive：**
- 完全相同的收益率：-192.33%
- 完全相同的交易次数：609
- 完全相同的所有指标
- **结论**: 参数设置实际无效，两者是相同策略
