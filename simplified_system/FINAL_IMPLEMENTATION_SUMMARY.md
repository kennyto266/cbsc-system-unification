### **核心算法**
1. **風險調整評分**:
   ```python
   score = Sharpe評分(40%) + 回報評分(30%) +
           回撤評分(20%) + 勝率評分(10%) - 風險懲罰
   ```

2. **多樣化比率計算**:
   ```python
   DR = N / (1 + (N-1) * avg_correlation)
   ```

3. **動態頭寸計算**:
   ```python
   頭寸 = min(波動率目標/當前波動率 * 基礎頭寸,
              最大頭寸限制) * 風險調整因子
   ```

- **容錯**: 缺失數據自動補全，格式智能識別
