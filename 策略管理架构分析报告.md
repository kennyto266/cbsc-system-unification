# 策略管理UI架构分析报告

## 概述

本报告详细分析了为CBSC量化交易系统实现的策略管理UI架构。该系统提供了完整的策略生命周期管理功能，包括创建、编辑、执行监控和性能分析。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5.x
- **状态管理**: Redux Toolkit + RTK Query
- **图表库**: Chart.js + react-chartjs-2
- **路由**: React Router v6
- **样式**: CSS-in-JS + Ant Design主题系统

### 2. 核心组件详解

#### 2.1 StrategyList组件
- **功能**: 策略列表展示，支持表格和卡片两种视图
- **特性**:
  - 多维度筛选（类型、状态、风险等级）
  - 搜索功能
  - 批量操作支持
  - 分页和排序
  - 实时状态更新

#### 2.2 StrategyCard组件
- **功能**: 策略卡片展示，提供快速预览
- **特性**:
  - 关键性能指标展示
  - 状态标签
  - 快速操作按钮
  - 进度条可视化

#### 2.3 StrategyForm组件
- **功能**: 策略创建和编辑表单
- **特性**:
  - 动态表单字段（根据策略类型）
  - 参数验证
  - 技术指标选择器（477种）
  - 风险管理配置
  - 表单预设模板

#### 2.4 StrategyDetails组件
- **功能**: 策略详细信息展示
- **特性**:
  - 多标签页组织（概览、性能、执行历史、风险分析）
  - 性能图表集成
  - 实时执行监控
  - 参数配置展示

### 3. 技术指标管理系统

支持477种技术指标，分类包括：

- **趋势指标**: SMA, EMA, MACD, DEMA等
- **动量指标**: RSI, CCI, Williams %R等
- **波动率指标**: ATR, Bollinger Bands, VIX等
- **成交量指标**: OBV, AD Line, MFI等
- **形态识别**: K线形态、价格形态等
- **自定义指标**: 支持用户自定义组合

### 4. API设计

使用RTK Query管理API调用，提供的功能包括：

- 策略CRUD操作
- 状态管理
- 执行历史查询
- 回测功能
- 性能指标计算
- 策略导入/导出
- 模板管理

### 5. 状态管理

- 使用Redux Toolkit进行全局状态管理
- RTK Query处理服务端状态
- 自定义Hooks封装业务逻辑
- 乐观更新策略
