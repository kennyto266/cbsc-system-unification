## 使用指南

### 快速開始
```bash
# 1. 進入監控目錄
cd monitoring

# 2. 一鍵部署
./scripts/deploy-monitoring-stack.sh

# 3. 訪問監控系統
# Prometheus: http://localhost:9090
# Grafana: http://localhost:3010 (admin/cbsc_admin_2025)
# AlertManager: http://localhost:9093
```

### 配置自定義告警
```bash
# 1. 編輯告警規則
vim prometheus/cbsc-alerts.yml

# 2. 重新加載配置
curl -X POST http://localhost:9090/-/reload

# 3. 驗證規則
curl http://localhost:9090/api/v1/rules
```

### 添加新監控目標
```bash
# 1. 編輯Prometheus配置
vim prometheus/production.yml

# 2. 添加新job配置
# - job_name: 'new-service'
#   static_configs:
#     - targets: ['new-service:8080']

# 3. 重啟Prometheus
docker restart cbsc-prometheus
```
