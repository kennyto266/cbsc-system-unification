## 💻 使用方式

### 基本使用

```python
from src.ui.enhanced_terminal_ui import get_ui, colored_text, status_indicator

ui = get_ui()
print(colored_text("彩色文本", 'cyan', bold=True))
print(status_indicator('success', '操作成功'))
```

### 進度條

```python
# 基本進度條
progress_id = ui.create_progress_bar(100, "處理中")
for i in range(100):
    ui.update_progress(progress_id, 1)

# 上下文管理器
with ui.progress_context(50, "計算中") as progress:
    for i in range(50):
        progress.update(1)
```

### 表格顯示

```python
headers = ["股票代碼", "價格", "漲跌幅"]
data = [["0700.HK", "368.20", "+2.15%"]]
ui.print_table(headers, data, title="股票行情", tablefmt="fancy_grid")
```

### ASCII圖表

```python
prices = [100, 102, 98, 105, 103]
chart = ui.create_ascii_chart(prices, title="價格走勢", width=60, height=10)
print(chart)
```

### 主題切換

```python
from src.ui.enhanced_terminal_ui import Theme

ui.set_theme(Theme.PROFESSIONAL)  # 切換到專業主題
```
