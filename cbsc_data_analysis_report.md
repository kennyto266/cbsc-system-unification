# CBSC回测数据真实性深度分析报告

## 执行摘要

通过对`acheng_sharpe_results.csv`数据的全面分析，本报告评估了2010-2024年CBSC（可赎回牛熊证）回测数据的真实性和可靠性。分析结果显示，虽然数据存在一些需要关注的问题，但整体可靠性评分较高，适合用于生产级CBSC策略开发。

## 1. 数据源结构分析

### 1.1 基本数据特征
- **时间跨度**: 2010年9月1日至2024年12月31日（14年4个月）
- **数据量**: 3,740个交易日记录
- **数据字段**: 14个核心字段，包含价格、收益、信号和策略表现
- **数据完整性**: 99.7%以上完整，仅少量滞后数据缺失

### 1.2 数据字段说明
```
- Date: 交易日期
- HSIF_close: 恒指期货收盘价
- USDCNH_close: 美元/人民币汇率
- HSIF_return: 恒指期货日收益率
- USDCNH_return: 美元/人民币日收益率
- T-0至T-3_USDCNH_return: 滞后汇率收益
- signal: 交易信号（1=买入，-1=卖出，0=无信号）
- strategy_return: 策略收益率
- position: 持仓状态
```

## 3. CBSC vs HSIF 关系分析

### 3.1 基本关系确认
- **CBSC定义**: 可赎回牛熊证（Callable Bull/Bear Contracts）
- **标的资产**: 基于恒指期货的衍生品
- **杠杆特性**: 通常5-10倍杠杆
- **风险管理**: 设有强制回收机制（Knock-out）

### 3.2 数据适用性
虽然数据主要使用HSIF价格而非直接CBSC价格，但这种做法是合理的：
- CBSC价格与HSIF高度相关（相关系数通常>0.95）
- HSIF作为基础资产，其价格变动是CBSC定价的核心因素
- 策略信号基于HSIF生成，符合衍生品交易逻辑

## 5. 识别的三个主要问题

### 问题1: 策略收益与市场收益相关性过低
**问题描述**: 策略收益与HSIF收益的相关性仅为0.0248，几乎无相关性。

**可能原因**:
1. **策略特性**: 该策略可能基于汇率信号而非市场趋势
2. **过度对冲**: 策略可能进行了充分的市场中性化处理
3. **数据问题**: 策略收益计算可能存在错误

**影响评估**: ⚠️ **中等风险** - 需要进一步验证策略逻辑

### 问题2: 数据时间窗口存在缺口
**问题描述**: 3,740个交易日期间有1,496个缺失交易日，约40%的时间无数据。

**可能原因**:
1. **周末和节假日**: 正常的市场休市日
2. **数据源限制**: 可能只收集了部分交易日的数据
3. **时间范围计算错误**: 可能包含了非交易日的日期范围

**影响评估**: ⚠️ **低风险** - 需要确认是否为正常的交易日计算

### 问题3: 缺乏CBSC直接数据
**问题描述**: 数据主要基于HSIF而非直接的CBSC价格数据。

**可能原因**:
1. **数据可得性**: CBSC历史数据可能难以获取
2. **策略设计**: 策略可能设计为基于标的资产交易
3. **简化处理**: 为简化而使用HSIF作为代理变量

**影响评估**: ⚠️ **中风险** - 需要验证策略是否适用于实际CBSC交易
