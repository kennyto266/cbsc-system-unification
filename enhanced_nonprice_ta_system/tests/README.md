## 🐛 故障排除

### 常見問題
1. **API連接失敗**: 檢查網絡連接和API端點可用性
2. **內存不足**: 減少並行度或測試數據大小
3. **導入錯誤**: 確保項目路徑正確和依賴完整
4. **權限錯誤**: 檢查文件系統權限

### 調試建議
- 使用詳細輸出模式：`python -v run_phase6_tests.py`
- 檢查日誌文件：查看詳細錯誤信息
- 運行單獨測試：隔離問題模塊
- 系統監控：檢查資源使用情況
