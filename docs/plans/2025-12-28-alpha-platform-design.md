## 1. 項目概述

### 1.1 目標

建立一個支持多種非價格數據源的 Alpha 策略快速驗證平台，使量化研究員能夠：

- 使用宏觀經濟、市場情緒、資金流向、另類數據尋找 Alpha
- 通過簡單統計因子快速驗證策略想法
- 在 Web Dashboard 或 Jupyter Notebook 中靈活研究
- 進行全市場多策略批量回測

### 1.2 核心特性

✅ **多數據源支持**: 宏觀、情緒、資金流、另類數據  
✅ **多頻率架構**: 保留原始數據時間維度  
✅ **統計因子庫**: 20+ 預設因子，易於理解  
✅ **高性能回測**: VectorBT 多進程，秒級響應  
✅ **雙界面**: Web Dashboard + Jupyter Notebook  
✅ **內存優先**: Redis 緩存，極速體驗

## 3. 數據適配層

### 3.1 數據源分類

| 類別 | 數據源 | 頻率 | 適配器 | 狀態 |
|------|--------|------|--------|------|
| 宏觀經濟 | GDP、CPI、失業率、利率 | 季度/月度 | EconomicDataAdapter | ✅ 已有 |
| 資金流向 | 北向資金流動、持倉 | 日度 | NorthboundFlowAdapter | ✅ 已有 |
| 市場情緒 | 新聞情緒、社交媒體 | 日度/實時 | SentimentDataAdapter | 🔄 新建 |
| 另類數據 | 衛星圖像、消費數據 | 不固定 | AlternativeDataAdapter | 🔄 新建 |

### 3.2 統一數據格式

```python
@dataclass
class AlphaSignal:
    """統一的 Alpha 信號格式"""
    timestamp: datetime
    symbol: str
    signal_value: float    # 信號值，範圍 [-1, 1]
    signal_type: str      # 信號類型：momentum, mean_reversion, etc.
    source: str           # 數據源標識
    confidence: float     # 信號置信度 [0, 1]
    metadata: Dict[str, Any] = None  # 額外元數據
```
