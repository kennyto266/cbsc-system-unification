### 2. 風險分析模塊增強 📊

**新增指標：**
- 卡爾瑪比率（Calmar Ratio）
- 詹森指數（Jensen's Alpha）
- 特雷諾比率（Treynor Ratio）
- 索提諾比率（Sortino Ratio）
- 最大回撤期分析
- 下行風險計算

**實現方案：**
```python
class EnhancedRiskMetrics:
    def calculate_all(self, returns, benchmark_returns=None):
        metrics = {
            'sharpe': self.sharpe_ratio(returns),
            'calmar': self.calmar_ratio(returns),
            'jensen': self.jensen_alpha(returns, benchmark_returns),
            'treynor': self.treynor_ratio(returns),
            'sortino': self.sortino_ratio(returns),
            'max_drawdown': self.max_drawdown(returns),
            'downside_deviation': self.downside_deviation(returns)
        }
        return metrics
```
