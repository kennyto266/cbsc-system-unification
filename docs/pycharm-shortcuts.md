## 🚀 最常用快捷键

### 基础操作
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl + Alt + S` | 打开设置 | 打开 PyCharm 设置窗口 |
| `Ctrl + Shift + A` | 查找操作 | 搜索所有菜单和操作 |
| `Alt + 1` | 项目窗口 | 打开/关闭项目文件树 |
| `Ctrl + E` | 最近文件 | 查看最近打开的文件 |
| `Ctrl + Shift + E` | 最近编辑位置 | 在最近编辑位置之间跳转 |

## 🛠️ 自定义快捷键

### 设置自定义快捷键
1. 打开设置：`Ctrl + Alt + S`
2. 导航到：`Keymap`
3. 找到要修改的操作
4. 双击 → `Add Keyboard Shortcut`
5. 按下想要的快捷键组合

### 常用的自定义设置
- **Git 提交**: `Ctrl + Enter`
- **Git 推送**: `Ctrl + Shift + Enter`
- **格式化选中代码**: `Ctrl + Alt + L`
