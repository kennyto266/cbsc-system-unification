## 4. 數據流向圖

```mermaid
sequenceDiagram
    participant C as Client
    participant API as API Router
    participant SVC as Service Layer
    participant REPO as Repository
    participant DB as Database
    participant CACHE as Cache

    C->>API: Create Strategy
    API->>SVC: StrategyService.create()
    SVC->>REPO: StrategyRepository.save()
    REPO->>DB: INSERT strategy
    DB-->>REPO: strategy_id
    REPO-->>SVC: Strategy object
    SVC->>CACHE: Cache strategy
    SVC-->>API: Strategy response
    API-->>C: 201 Created

    C->>API: Get Strategy
    API->>SVC: StrategyService.get()
    SVC->>CACHE: Try cache first
    alt Cache hit
        CACHE-->>SVC: Cached strategy
    else Cache miss
        SVC->>REPO: StrategyRepository.find()
        REPO->>DB: SELECT * FROM strategies
        DB-->>REPO: Strategy data
        REPO-->>SVC: Strategy object
        SVC->>CACHE: Cache result
    end
    SVC-->>API: Strategy response
    API-->>C: 200 OK
```

## 5. 依賴注入配置

### 5.1 FastAPI依賴注入

```python
# __init__.py
from fastapi import FastAPI
from .services import (
    StrategyService,
    ExecutionService,
    PerformanceService,
    TemplateService,
    NotificationService
)
from .repositories import (
    StrategyRepository,
    SignalRepository,
    PerformanceRepository
)

# 依賴注入容器
class DIContainer:
    def __init__(self):
        # Repository層
        self.strategy_repository = StrategyRepository()
        self.signal_repository = SignalRepository()
        self.performance_repository = PerformanceRepository()

        # Service層
        self.template_service = TemplateService(self.strategy_repository)
        self.performance_service = PerformanceService(self.performance_repository)
        self.strategy_service = StrategyService(
            repository=self.strategy_repository,
            performance_service=self.performance_service,
            template_service=self.template_service
        )
        self.execution_service = ExecutionService(
            strategy_service=self.strategy_service,
            signal_repository=self.signal_repository
        )
        self.notification_service = NotificationService()

    def get_strategy_service(self) -> StrategyService:
        return self.strategy_service

    def get_execution_service(self) -> ExecutionService:
        return self.execution_service

# 全局容器
container = DIContainer()

# FastAPI依賴
async def get_strategy_service() -> StrategyService:
    return container.get_strategy_service()
```
