    self.universe = ['AAPL', 'MSFT', 'GOOGL']
    
    # 設置均線參數
    self.short_window = 10
    self.long_window = 30

[畫面：處理數據方法實現]

在handle_data方法中實現交易邏輯：
```python
def handle_data(self, context, data):
    for stock in self.universe:
        # 獲取歷史價格
        prices = data.history(stock, 'close', self.long_window)
        
        # 計算移動平均線
        short_ma = prices.rolling(self.short_window).mean()[-1]
        long_ma = prices.rolling(self.long_window).mean()[-1]
        
        # 獲取當前持倉
        current_position = context.positions.get(stock, 0)
        
        # 買入條件：金叉且無持倉
        if short_ma > long_ma and current_position == 0:
            # 計算目標倉位（10%）
            target_value = context.portfolio_value * 0.1
            shares = int(target_value / data.current[stock].close)
            
            if shares > 0:
                self.order(stock, shares)
        
        # 賣出條件：死叉且有持倉
        elif short_ma < long_ma and current_position > 0:
            self.order_target(stock, 0)
```

from cbsc.indicators import (
    SMA, EMA, RSI, MACD,

class TechnicalStrategy(Strategy):
    def initialize(self):
        # 初始化指標
        self.sma_short = SMA(period=10)
        self.sma_long = SMA(period=30)
        self.rsi = RSI(period=14)

[畫面：趨勢指標應用]

移動平均線組合：
```python
def handle_trend_signals(self, context, data, stock):
    # 計算均線值
    short_ma = self.sma_short.calculate(data.history(stock, 'close', 30))
    long_ma = self.sma_long.calculate(data.history(stock, 'close', 60))
    
    # 趨勢判斷
    if short_ma > long_ma:
        trend = 'uptrend'
    elif short_ma < long_ma:
        trend = 'downtrend'
    else:
        trend = 'sideways'
    
    return trend
```

[畫面：動量指標應用]

RSI超買超賣策略：
```python
def handle_momentum_signals(self, context, data, stock):
    # 計算RSI
    prices = data.history(stock, 'close', 50)
    rsi_value = self.rsi.calculate(prices)
    
    # RSI信號
    if rsi_value < 30:  # 超賣
        return 'oversold'
    elif rsi_value > 70:  # 超買
        return 'overbought'
    else:
        return 'neutral'
```

[畫面：波動率指標應用]

布林帶突破策略：
```python
def handle_volatility_signals(self, context, data, stock):
    # 計算布林帶
    prices = data.history(stock, 'close', 30)
    bb_upper, bb_middle, bb_lower = self.bb.calculate(prices)
    current_price = data.current[stock].close
    
    # 布林帶信號
    if current_price > bb_upper:
        return 'breakout_up'
    elif current_price < bb_lower:
        return 'breakout_down'
    else:
        return 'within_bands'
```

[畫面：多指標組合策略]

組合多個指標生成更可靠的信號：
```python
def generate_signal(self, context, data, stock):
    # 獲取各類信號
    trend = self.handle_trend_signals(context, data, stock)
    momentum = self.handle_momentum_signals(context, data, stock)
    volatility = self.handle_volatility_signals(context, data, stock)
    
    # 信號組合邏輯
    if trend == 'uptrend' and momentum == 'oversold':
        return 'BUY'
    elif trend == 'downtrend' or momentum == 'overbought':
        return 'SELL'
    else:
        return 'HOLD'
```

### 視頻7：參數優化技巧（15分鐘）

**腳本**：

```
主持人：大家好！今天我們將學習如何對策略參數進行優化，以找到最佳的參數組合。

[畫面：進入參數優化界面]

在策略詳情頁面，點擊"參數優化"按鈕，進入優化配置界面。

[畫面：設置優化參數]

首先，選擇要優化的參數：
- 均線短期週期：5-20
- 均線長期週期：20-50
- RSI週期：10-20
- 止損比例：2%-10%

[畫面：優化方法選擇]

系統提供三種優化方法：

1. 網格搜索（Grid Search）
   - 遍歷所有參數組合
   - 確保找到全局最優解
   - 計算量較大

2. 隨機搜索（Random Search）
   - 隨機選擇參數組合
   - 計算效率較高
   - 可能漏過最優解

3. 貝葉斯優化（Bayesian Optimization）
   - 智能選擇參數
   - 收斂速度快
   - 適合高維參數

[畫面：優化目標設置]

選擇優化目標：
- 最大夏普比率（推薦）
- 最大收益率

2. 並行執行回測
