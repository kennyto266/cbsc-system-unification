## 系統概述

CBSC量化交易系統是一個專業的量化策略管理平台，提供：

- **策略開發**：支持技術分析、基本面分析等多種策略類型
- **回測分析**：基於歷史數據的量化回測功能
- **實時交易**：模擬交易和實盤交易支持
- **風險管理**：全面的風險控制和監控工具
- **數據服務**：提供完整的市場數據和經濟指標

## 策略管理

### 創建策略

#### 使用策略編輯器

1. 進入「策略」模塊
2. 點擊「新建策略」
3. 選擇策略模板或從零開始

**策略配置步驟：**

1. **基本信息**
   ```
   策略名稱：移動平均線交叉策略
   策略描述：基於快慢移動平均線交叉的交易策略
   策略類型：技術分析
   ```

2. **參數設置**
   ```
   - 短期均線週期：10
   - 長期均線週期：30
   - 交易標的：AAPL, MSFT, GOOGL
   - 時間週期：1日
   ```

3. **風險控制**
   ```
   - 單筆最大倉位：10%
   - 總倉位上限：80%
   - 止損比例：5%
   - 止盈比例：20%
   ```

4. **執行設置**
   ```
   - 執行頻率：每日
   - 開盤前執行：是
   - 收盤前檢查：是
   ```

#### 使用策略代碼

支持Python策略編寫：

```python
from cbsc.strategy import Strategy, Data, Context

class MACrossStrategy(Strategy):
    def initialize(self):
        # 初始化參數
        self.short_period = 10
        self.long_period = 30
        
        # 訂閱數據
        self.symbols = ["AAPL", "MSFT", "GOOGL"]
        
    def handle_data(self, context, data):
        for symbol in self.symbols:
            # 獲取歷史數據
            prices = data.history(symbol, "close", self.long_period)
            
            # 計算移動平均線
            short_ma = prices.rolling(self.short_period).mean()[-1]
            long_ma = prices.rolling(self.long_period).mean()[-1]
            
            # 交叉判斷
            if short_ma > long_ma and context.positions[symbol] == 0:
                # 金叉買入
                self.order_target_percent(symbol, 0.1)
            elif short_ma < long_ma and context.positions[symbol] > 0:
                # 死叉賣出
                self.order_target_percent(symbol, 0)
```

### 策略類型

#### 1. 技術分析策略
- **移動平均策略**：基於均線交叉
- **動量策略**：RSI、MACD等指標
- **突破策略**：支撐阻力位突破
- **趨勢跟蹤**：跟蹤市場趨勢

#### 2. 基本面策略
- **價值投資**：基於財務數據
- **成長投資**：關注成長性股票
- **質量投資**：優質公司篩選
- **股息策略**：高股息率選股

#### 3. 量化策略
- **統計套利**：價格統計關係
- **配對交易**：相關性交易
- **事件驅動**：基於特定事件
- **機器學習**：AI驅動策略

## 常見問題

### Q1：忘記密碼怎麼辦？

A：點擊登錄頁的「忘記密碼」，輸入註冊郵箱，系統會發送重置密碼鏈接。

### Q2：如何切換環境？

A：在設置菜單中選擇「環境設置」，可以切換：
- 生產環境（實盤）
- 測試環境（模擬）
- 開發環境（測試）

### Q3：策略執行失敗怎麼辦？

A：
1. 檢查策略參數是否正確
2. 確認賬戶資金充足
3. 檢查網絡連接
4. 聯繫技術支持

### Q4：數據更新頻率？

A：
- 實時行情：15秒更新
- 分鐘數據：每分鐘更新
- 日線數據：每日收盤後更新
- 基本面數據：每季度更新

### Q5：如何提高回測準確性？

A：
1. 使用更長的歷史數據
2. 包含滑點和手續費
3. 進行樣本外測試
4. 多參數敏感性分析

### Q6：支持哪些經紀商？

A：支持主流經紀商：
- Interactive Brokers
- 富途牛牛
- 老虎證券
- 華泰證券
- 更多支持開發中...
