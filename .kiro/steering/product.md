# CBSC 量化交易策略管理系統

基於 Claude Code PM 構建的企業級量化交易策略管理平台，為個人獨立使用優化。

## Core Capabilities

### 策略管理
- **多元策略支持**: 技術指標、基本面、情緒分析、宏觀經濟數據驅動策略
- **回測引擎**: 支援標準回測、風險管理回測、壓力測試、蒙地卡羅模擬
- **即時監控**: 策略執行實時監控與動態風險調整

### 用戶系統
- **多因子認證**: 郵箱、簡訊、Google Authenticator
- **權限控制**: 基於角色的訪問控制(RBAC)
- **個人中心**: 資料管理、安全設置、通知偏好

### 數據分析
- **技術分析圖表**: Chart.js + Plotly.js 雙引擎支援
- **經濟指標**: HKMA 宏觀經濟數據整合
- **情緒分析**: 市場情緒指標與新聞情感分析

## Target Use Cases

### 個人量化交易者
- 策略開發與回測驗證
- 多市場數據分析（港股、美股、加密貨幣）
- 風險管理與投資組合優化

### 小型投資團隊
- 協作式策略開發
- 統一的策略執行平台
- 合規審計與性能追蹤

## Value Proposition

**AI驅動的開發體驗**: 基於 Claude Code PM 的規範驅動開發流程，確保每行代碼都可追溯到需求

**漸進式集成**: 與現有 CBSC 系統無縫整合，支持平滑遷移和版本兼容

**企業級架構**: 微服務設計、容器化部署、監控告警完整鏈路

**開源友好**: 完整的文檔、測試覆蓋、可擴展的插件體系
