```

├── src/                    # 核心源代码
│   ├── cbsc/              # CBSC策略模块
│   ├── strategies/        # 交易策略
│   ├── backtest/         # 回测引擎
│   ├── data/             # 数据处理
│   ├── indicators/       # 技术指标
│   └── utils/            # 工具函数

│   ├── integration/      # 集成测试
│   └── e2e/              # 端到端测试

├── scripts/               # 脚本工具
├── docs/                  # 文档
├── data/                  # 数据文件
├── config/                # 配置文件
├── reports/               # 生成的报告
├── cache/                 # 缓存文件
└── archives/              # 归档文件
```

## **🗑️ 建议删除的文件类型**

### **立即删除**
- 所有.cache目录
- 所有.temp文件
- 所有.py.backup文件
- 重复的分析报告JSON文件
- 临时截图和快照

### **移动到archives**
- 旧的优化结果
- 历史分析报告
- 版本对比文件
- 实验性代码

### **保留在根目录**
- README.md
- requirements.txt
- setup.py
- .gitignore
- 主要启动脚本
- 配置文件
