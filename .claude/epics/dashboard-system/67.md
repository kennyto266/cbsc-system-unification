## Task Description

Develop and implement real-time strategy performance widgets for the dashboard, providing users with instant visibility into their trading strategies' status and performance metrics.

## Scope

### In Scope
- Strategy status widget with live updates
- Performance metrics visualization (returns, Sharpe ratio, drawdown)
- Portfolio overview with asset allocation chart
- Real-time data integration via WebSocket
- Mobile-responsive widget layouts
- Interactive charts and drill-down capabilities

### Out of Scope
- Historical data analysis (covered by Task 006)
- Strategy creation/modification interfaces
- Order execution functionality
- Risk management settings

## Requirements

### Functional Requirements

#### Strategy Status Widget
- **R1**: Display active/inactive status for all user strategies
- **R2**: Show last execution time and next scheduled run
- **R3**: Real-time status updates via WebSocket
- **R4**: Quick toggle for enabling/disabling strategies

#### Performance Metrics Widget
- **R5**: Display key performance indicators:
  - Total return (percentage and absolute)
  - Sharpe ratio with risk-free rate configuration
  - Maximum drawdown with recovery period
  - Win rate and profit factor
  - Average trade duration
- **R6**: Configurable time periods (1D, 1W, 1M, 3M, 6M, 1Y, All)
- **R7**: Benchmark comparison capability

#### Portfolio Overview Widget
- **R8**: Asset allocation pie chart with dynamic updates
- **R9**: Sector/category breakdown with drill-down
- **R10**: Top 10 holdings with weight percentages
- **R11**: Cash position and available margin
- **R12**: Rebalancing suggestions based on target allocation

### Non-Functional Requirements

#### Performance
- **N1**: Widget updates within 500ms of data change
- **N2**: Initial load time under 2 seconds
- **N3**: Efficient WebSocket reconnection handling

#### Usability
- **N4**: Intuitive visual design with clear metric definitions
- **N5**: Tooltips and help text for all metrics
- **N6**: Dark/light theme support with consistent styling

## Implementation Details

### Frontend Components

```typescript
// Strategy Status Widget
interface StrategyStatus {
  id: string;
  name: string;
  status: 'active' | 'inactive' | 'error';
  lastExecution: Date;
  nextExecution?: Date;
  dailyPnL: number;
  error?: string;
}

// Performance Metrics Widget
interface PerformanceMetrics {
  period: string;
  totalReturn: number;
  sharpeRatio: number;
  maxDrawdown: number;
  winRate: number;
  profitFactor: number;
  avgTradeDuration: number;
}

// Portfolio Overview Widget
interface PortfolioAllocation {
  totalValue: number;
  cash: number;
  invested: number;
  assets: AssetAllocation[];
  sectors: SectorAllocation[];
}
```

- Subscribe to strategy status updates
- Implement automatic reconnection with exponential backoff
- Handle message queuing during disconnection

### Chart Implementation
- Use Chart.js for responsive charts

- Add animation effects for data updates
- Support chart export functionality
