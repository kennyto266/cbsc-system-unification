# Task 002: Responsive Grid System and Widget Management

## Description

實現Dashboard的響應式網格系統和可拖拽的小部件（Widget）管理功能，支持動態佈局調整和個性化配置。

## Requirements

### 1. 響應式網格系統
- 基於CSS Grid的彈性佈局
- 支持12欄網格系統
- 自適應不同屏幕尺寸
- 網格間隙和對齊配置

### 2. Widget系統
- 可拖拽移動的Widget容器
- Widget大小調整功能
- Widget的最小/最大尺寸限制
- Widget的添加/刪除操作

### 3. 預定義Widget類型
- 策略概覽Widget
- 性能指標Widget
- 回測結果Widget
- 實時監控Widget
- 新聞公告Widget

### 4. 佈局管理
- 保存/加載用戶佈局配置
- 預設佈局模板
- 一鍵重置佈局
- 佈局導入/導出

### 數據結構
```typescript
// Widget配置
interface WidgetConfig {
  id: string;
  type: WidgetType;
  title: string;
  x: number;
  y: number;
  w: number;
  h: number;
  minW?: number;
  minH?: number;
  maxW?: number;
  maxH?: number;
  isDraggable?: boolean;
  isResizable?: boolean;
  config?: Record<string, any>;
}

// 佈局配置
interface LayoutConfig {
  name: string;
  widgets: WidgetConfig[];
  breakpoints: {
    lg: number;
    md: number;
    sm: number;
    xs: number;
    xxs: number;
  };
}
```

### 核心組件實現
```typescript
// GridSystem組件
const GridSystem: React.FC<GridSystemProps> = ({
  layout,
  onLayoutChange,
  widgets,
  children
}) => {
  return (
    <ResponsiveGridLayout
      className="layout"
      layouts={layout}
      onLayoutChange={onLayoutChange}
      breakpoints={{lg: 1200, md: 996, sm: 768, xs: 480, xxs: 0}}
      cols={{lg: 12, md: 10, sm: 6, xs: 4, xxs: 2}}
      rowHeight={60}
      isDraggable={true}
      isResizable={true}
    >
      {children}
    </ResponsiveGridLayout>
  );
};

// Widget容器組件
const WidgetContainer: React.FC<WidgetContainerProps> = ({
  widget,
  onEdit,
  onDelete,
  onResize,
  children
}) => {
  return (
    <div className="widget-container bg-white rounded-lg shadow-md p-4">
      <WidgetHeader widget={widget} onEdit={onEdit} onDelete={onDelete} />
      <WidgetContent>{children}</WidgetContent>
      <WidgetResizeHandle onResize={onResize} />
    </div>
  );
};
```
