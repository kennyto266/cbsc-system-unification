# Task #68: Analytics API Endpoints

## Overview

开发策略性能分析API端点，为前端Widget提供实时和历史的性能数据。包括策略指标计算、历史数据聚合、投资组合分析等功能。

## Technical Requirements

### API端点列表

1. **GET /api/v1/analytics/strategies/{strategy_id}/performance**
   - 获取策略性能指标
   - 支持时间范围参数
   - 返回实时和历史数据

2. **GET /api/v1/analytics/portfolio/{user_id}/overview**
   - 获取用户投资组合概览
   - 资产配置分析
   - 风险指标计算

3. **GET /api/v1/analytics/strategies/{strategy_id}/returns**
   - 获取策略收益率曲线
   - 支持不同时间粒度（日线/周线/月线）
   - 基准比较

4. **GET /api/v1/analytics/reports/summary**
   - 生成策略执行报告
   - 支持PDF/Excel导出
   - 定时报告生成

5. **WebSocket /ws/analytics/realtime**
   - 实时数据推送
   - 策略状态更新
   - 性能指标变化

### 核心模型设计
```python
# models/analytics.py
class StrategyPerformance(Base):
    __tablename__ = "strategy_performance"

    id = Column(UUID, primary_key=True, default=uuid4)
    strategy_id = Column(UUID, ForeignKey("strategies.id"))
    date = Column(DateTime, index=True)

    # 收益指标
    daily_return = Column(Numeric(10, 4))
    cumulative_return = Column(Numeric(10, 4))

    # 风险指标
    volatility = Column(Numeric(10, 4))
    sharpe_ratio = Column(Numeric(10, 4))
    sortino_ratio = Column(Numeric(10, 4))
    max_drawdown = Column(Numeric(10, 4))
    var_95 = Column(Numeric(10, 4))

    # 交易统计
    total_trades = Column(Integer)
    winning_trades = Column(Integer)
    win_rate = Column(Numeric(5, 4))
    profit_factor = Column(Numeric(10, 4))

    created_at = Column(DateTime, default=datetime.utcnow)
```

### API Schema定义
```python
# schemas.py
from pydantic import BaseModel
from datetime import date, datetime
from typing import List, Optional

class PerformanceMetrics(BaseModel):
    strategy_id: str
    total_return: float
    annualized_return: float
    volatility: float
    sharpe_ratio: float
    max_drawdown: float
    win_rate: float
    profit_factor: float

    class Config:
        from_attributes = True

class PortfolioOverview(BaseModel):
    user_id: int
    total_value: float
    cash_balance: float
    invested_amount: float
    positions: List[Position]
    performance: PerformanceMetrics

class TimeSeriesData(BaseModel):
    date: date
    value: float
    benchmark: Optional[float] = None

class ReturnsData(BaseModel):
    strategy_id: str
    data: List[TimeSeriesData]
    period: str  # daily, weekly, monthly
```
