# Task: SR和MDD圖表實現

## Description

基於Chart.js庫實現策略Sharpe比率(SR)和最大回撤(MDD)的可視化圖表。開發專業的金融圖表組件，包括條形圖、折線圖和雷達圖等多種圖表類型，實現策略表現的直觀展示。支持實時數據更新、交互式操作和響應式設計，為個人策略管理提供專業級的數據可視化功能。

## Acceptance Criteria
- [ ] 集成Chart.js庫，配置基礎圖表環境
- [ ] 實現Sharpe比率條形圖，支持多策略對比顯示
- [ ] 開發最大回撤折線圖，顯示歷史回撤趨勢
- [ ] 創建策略表現雷達圖，多維度展示策略指標
- [ ] 實現圖表交互功能，包括懸停提示、點擊事件
- [ ] 支持圖表主題切換（淺色/深色模式）
- [ ] 添加圖表數據標籤和圖例說明
- [ ] 實現圖表響應式布局，適配不同屏幕尺寸
- [ ] 圖表渲染時間控制在1秒以內

### SR圖表實現
```javascript
// Sharpe比率條形圖
class SharpeRatioChart {
  constructor(containerId) {
    this.chart = new Chart(document.getElementById(containerId), {
      type: 'bar',
      data: {
        labels: [], // 策略名稱
        datasets: [{
          label: 'Sharpe比率',
          data: [], // SR數值
          backgroundColor: [], // 根據數值設置顏色
          borderColor: [],
          borderWidth: 2
        }]
      },
      options: {
        scales: {
          y: {
            beginAtZero: true,
            title: { display: true, text: 'Sharpe比率' }
          }
        },
        plugins: {
          tooltip: {
            callbacks: {
              label: function(context) {
                return `SR: ${context.parsed.y.toFixed(2)}`;
              }
            }
          }
        }
      }
    });
  }
}
```

### MDD圖表實現
```javascript
// 最大回撤折線圖
class MaxDrawdownChart {
  constructor(containerId) {
    this.chart = new Chart(document.getElementById(containerId), {
      type: 'line',
      data: {
        labels: [], // 時間標籤
        datasets: [] // 多策略回撤數據
      },
      options: {
        scales: {
          y: {
            beginAtZero: true,
            max: 1,
            title: { display: true, text: '最大回撤 (%)' },
            ticks: {
              callback: function(value) {
                return (value * 100).toFixed(1) + '%';
              }
            }
          }
        },
        elements: {
          line: { tension: 0.4 }, // 曲線平滑度
          point: { radius: 3 }
        }
      }
    });
  }
}
```

### 數據更新機制
```javascript
// 圖表數據更新
class ChartManager {
  updateCharts(performanceData) {
    this.updateSharpeChart(performanceData);
    this.updateMDDChart(performanceData);
    this.updateRadarChart(performanceData);
  }

  animateChartUpdate(chart, newData) {
    chart.data.datasets[0].data = newData;
    chart.update('active'); // 動畫更新
  }
}
```
