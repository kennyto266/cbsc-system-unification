# Task #69: Alert System and Notifications

## Overview

实现一个全面的通知和告警系统，支持多种通知渠道（浏览器推送、邮件、短信），提供灵活的告警规则配置，确保用户能够及时收到重要的事件通知。

## Technical Requirements

### 功能模块

1. **告警规则引擎**
   - 阈值告警（收益率、回撤等）
   - 事件告警（策略启动/停止/错误）
   - 自定义规则配置
   - 规则优先级管理

2. **通知渠道管理**
   - 浏览器推送通知
   - 邮件通知（SMTP）
   - 短信通知（可选）
   - 站内消息
   - WebSocket实时推送

3. **通知配置界面**
   - 用户偏好设置
   - 通知模板管理
   - 免打扰时段配置
   - 通知历史查看

4. **通知中心**
   - 未读消息计数
   - 消息分类和筛选
   - 批量操作（标记已读/删除）
   - 消息详情查看

### 数据模型设计

```python
# models/alerts.py
class AlertRule(Base):
    __tablename__ = "alert_rules"

    id = Column(UUID, primary_key=True, default=uuid4)
    user_id = Column(Integer, ForeignKey("users.id"))
    name = Column(String(100), nullable=False)
    description = Column(Text)

    # 规则配置
    rule_type = Column(Enum(AlertType))  # THRESHOLD, EVENT, CUSTOM
    conditions = Column(JSON)  # 告警条件配置
    severity = Column(Enum(Severity))  # LOW, MEDIUM, HIGH, CRITICAL

    # 触发配置
    enabled = Column(Boolean, default=True)
    cooldown_minutes = Column(Integer, default=60)  # 冷却时间

    created_at = Column(DateTime, default=datetime.utcnow)
    updated_at = Column(DateTime, default=datetime.utcnow, onupdate=datetime.utcnow)

class Notification(Base):
    __tablename__ = "notifications"

    id = Column(UUID, primary_key=True, default=uuid4)
    user_id = Column(Integer, ForeignKey("users.id"))
    type = Column(Enum(NotificationType))
    title = Column(String(200))
    content = Column(Text)

    # 渠道配置
    channels = Column(JSON)  # ["browser", "email", "sms"]
    status = Column(Enum(NotificationStatus))  # PENDING, SENT, FAILED

    # 元数据
    metadata = Column(JSON)  # 关联的策略ID、交易ID等
    read_at = Column(DateTime)
    created_at = Column(DateTime, default=datetime.utcnow)
```

### 告警规则示例

```python
# services/alert_engine.py
class AlertEngine:
    def evaluate_rules(self, strategy_id: str, metrics: dict):
        """评估告警规则"""
        rules = self.get_active_rules(strategy_id)

        for rule in rules:
            if self.should_trigger(rule, metrics):
                self.create_alert(rule, strategy_id, metrics)

    def should_trigger(self, rule: AlertRule, metrics: dict):
        """判断是否触发告警"""
        if rule.rule_type == AlertType.THRESHOLD:
            return self.check_threshold(rule, metrics)
        elif rule.rule_type == AlertType.EVENT:
            return self.check_event(rule, metrics)

    def check_threshold(self, rule: AlertRule, metrics: dict):
        """阈值告警检查"""
        conditions = rule.conditions
        metric = metrics.get(conditions['metric'])
        threshold = conditions['threshold']
        operator = conditions['operator']

        if operator == 'gt':
            return metric > threshold
        elif operator == 'lt':
            return metric < threshold
        # ... 其他操作符
```
