# Task: 基礎頁面結構和樣式開發

## Description

創建個人策略管理Dashboard的基礎HTML結構和CSS樣式，建立響應式布局框架，為後續組件開發提供穩固的頁面基礎。基於現有系統設計，採用簡化技術棧（HTML5 + CSS3 + Vanilla JavaScript），實現1920x1080及以上分辨率的最佳顯示效果。

## Acceptance Criteria
- [x] 創建完整的HTML5頁面結構，包含DOCTYPE、meta標籤、語言設置
- [x] 實現響應式網格布局，支持桌面(1920x1080+)和平板設備
- [x] 設計清晰的導航區域，包含標題和刷新按鈕
- [x] 建立雙列內容布局：左側策略表現區，右側策略管理區
- [x] 實現統一的色彩方案和字體樣式
- [x] 添加適當的間距、陰影和邊框美化效果
- [x] 確保在不同瀏覽器（Chrome、Firefox）中的兼容性
- [x] 頁面加載時間控制在2秒以內

### HTML結構設計
- **語言和編碼**：`lang="zh-CN"`, `charset="UTF-8"`
- **響應式viewport**：`width=device-width, initial-scale=1.0`
- **主容器**：固定最大寬度1600px，居中顯示
- **網格布局**：CSS Grid或Flexbox實現雙列布局
- **組件預留**：為圖表容器、策略列表、控制面板預留空間

### CSS樣式規範
- **色彩方案**：
  - 主色調：`#2c3e50` (深藍灰)
  - 強調色：`#3498db` (藍色)
  - 成功色：`#27ae60` (綠色)
  - 警告色：`#f39c12` (橙色)
  - 危險色：`#e74c3c` (紅色)
- **字體**：系統字體棧 `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, Arial`
- **間距系統**：8px基礎網格 (8px, 16px, 24px, 32px)
- **陰影效果**：`box-shadow: 0 2px 8px rgba(0,0,0,0.1)`

### 響應式設計
- **桌面優化**：1920x1080及以上分辨率優化
- **平板支持**：768px-1024px寬度適配
- **手機兼容**：320px-768px寬度基礎支持

### 性能優化
- **CSS壓縮**：生產環境CSS壓縮
- **圖片優化**：使用WebP格式和適當尺寸
- **字體優化**：使用系統字體避免加載延遲
