**1. Data Source Failures**
```python
async def get_hkma_data_with_fallback(indicator_type: str):
    try:
        # Try primary HKMA API
        data = await hkma_api.get_data(indicator_type)
    except HKMAAPIError:
        # Fallback to cached data
        data = await get_cached_data(indicator_type)
    except Exception:
        # Final fallback to mock data
        data = generate_mock_data(indicator_type)
    return data
```
