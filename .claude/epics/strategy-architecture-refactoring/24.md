# Task: 缓存集成与监控

## Description
将CacheManager集成到所有策略管理器中，替换现有的分散缓存，并实现缓存监控指标。

## Acceptance Criteria
- [ ] 替换所有Manager中的分散缓存
- [ ] 缓存命中率>75%
- [ ] 内存使用<1GB且稳定
- [ ] Prometheus指标导出
- [ ] Grafana仪表板配置
- [ ] 缓存一致性验证通过
- [ ] 压力测试通过

## Technical Details
- **集成范围**:
  - StrategyManager - 策略基础信息缓存
  - PersonalStrategyManager - 用户数据缓存
  - CBSCStrategyManager - 性能数据缓存
- **监控指标**:
  - cache_hits_total - 缓存命中次数
  - cache_misses_total - 缓存未命中次数
  - cache_hit_rate - 缓存命中率
  - cache_memory_bytes - 内存使用量
  - cache_operation_duration_seconds - 操作延迟
- **集成步骤**:
  1. 注入CacheManager依赖
  2. 替换所有self.cache调用
  3. 添加Prometheus metrics
  4. 配置Grafana仪表板
  5. 运行压力测试
