## Technical Details
- **核心类设计**:
  ```python
  class WebSocketConnectionPool:
      - connections: Dict[str, ConnectionInfo]
      - user_connections: Dict[int, List[str]]
      - subscriptions: Dict[str, Set[str]]
      - add_connection() / remove_connection()
      - subscribe() / unsubscribe()
      - broadcast() / send_to_user()
      - 健康检查后台任务
  ```
- **关键功能**:
  - 连接数限制和验证
  - 空闲连接超时清理（5分钟）
  - 心跳机制（30秒间隔）
  - 订阅管理和消息路由
- **文件结构**:
  - `src/services/websocket_pool.py` - 连接池实现
  - `src/api/strategies/websocket.py` - FastAPI集成
  - `tests/test_websocket_pool.py` - 单元测试
  - `tests/test_websocket_e2e.py` - E2E测试
