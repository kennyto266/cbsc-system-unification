## Technical Details
- **核心类设计**:
  ```python
  class CacheManager:
      - memory_cache: TTLCache (L1)
      - redis_client: Redis (L2)
      - strategies: Dict[str, CacheStrategy]
      - get(key) / set(key, value, ttl)
      - clear_pattern(pattern)
  ```
- **缓存策略**:
  - strategy数据：5分钟，L2缓存
  - performance数据：30秒，L1缓存
  - user数据：1分钟，L1+L2缓存
  - config数据：10分钟，L2缓存
- **文件位置**:
  - `src/services/cache_manager.py` - 核心实现
  - `src/services/cache_strategy.py` - 策略定义
  - `tests/test_cache_manager.py` - 单元测试
