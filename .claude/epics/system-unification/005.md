# Task: Migrate Core CBSC Strategy Management APIs

## Description
将现有CBSC量化策略管理功能迁移到新的统一架构中，包括策略创建、编辑、执行、监控等核心功能。保持与现有系统的兼容性，确保数据平滑迁移。

## Acceptance Criteria
- [ ] 策略CRUD API (创建、读取、更新、删除)
- [ ] 策略参数配置和验证
- [ ] 策略执行引擎接口
- [ ] 实时策略监控和状态更新
- [ ] 策略性能指标计算
- [ ] 与现有CBSC数据格式兼容
- [ ] 策略模板和预设配置

## Technical Details
- **API框架**: FastAPI + Pydantic数据验证
- **策略引擎**: 保留现有核心算法，统一接口
- **数据存储**: PostgreSQL + Redis缓存
- **实时通信**: WebSocket连接状态更新
- **兼容层**: 适配现有数据格式

**API端点设计**:
```
GET    /api/strategies              # 获取策略列表
POST   /api/strategies              # 创建新策略
GET    /api/strategies/{id}         # 获取策略详情
PUT    /api/strategies/{id}         # 更新策略
DELETE /api/strategies/{id}         # 删除策略
POST   /api/strategies/{id}/execute # 执行策略
GET    /api/strategies/{id}/status  # 获取策略状态
POST   /api/strategies/{id}/stop    # 停止策略执行
GET    /api/strategies/templates    # 获取策略模板
```

**策略数据结构**:
```python
class Strategy(BaseModel):
    name: str
    description: str
    type: StrategyType  # CBSC, Technical, etc.
    parameters: Dict[str, Any]
    status: StrategyStatus
    created_at: datetime
    updated_at: datetime
```
