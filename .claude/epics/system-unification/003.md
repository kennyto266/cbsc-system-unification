### API端點

#### 認證端點 (src/auth/api.py)
```
POST   /api/auth/register              # 用戶註冊
POST   /api/auth/login                 # 用戶登錄
POST   /api/auth/logout                # 用戶登出
POST   /api/auth/refresh               # 刷新令牌
POST   /api/auth/verify-email          # 驗證郵箱
POST   /api/auth/password/reset        # 密碼重置請求
POST   /api/auth/password/reset/confirm # 確認密碼重置
POST   /api/auth/password/change       # 修改密碼
GET    /api/auth/me                    # 獲取當前用戶
PUT    /api/auth/me                    # 更新用戶信息
```

#### MFA端點 (src/api/mfa.py)
```
POST   /api/mfa/setup                 # 設置MFA
POST   /api/mfa/verify                # 驗證MFA
DELETE /api/mfa/disable               # 禁用MFA
GET    /api/mfa/qr-code               # 獲取二維碼
POST   /api/mfa/backup-codes          # 生成備份碼
```

#### 用戶管理端點 (src/api/users/v2/)
```
GET    /api/users                     # 用戶列表
POST   /api/users                     # 創建用戶
GET    /api/users/{id}                # 用戶詳情
PUT    /api/users/{id}                # 更新用戶
DELETE /api/users/{id}                # 刪除用戶
GET    /api/users/{id}/sessions       # 用戶會話
GET    /api/users/{id}/devices        # 用戶設備
GET    /api/users/{id}/login-history  # 登錄歷史
```
