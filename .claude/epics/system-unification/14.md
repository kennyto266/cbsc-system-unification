# Task: Setup Development Environment and API Gateway

## Description
建立统一的项目开发环境，包括Docker容器化配置、API网关基础框架，以及统一的项目结构。这将为后续所有开发工作提供一致的基础设施。

## Acceptance Criteria
- [ ] 一键启动开发环境脚本 (`docker-compose up -d`)
- [ ] API网关基础框架，支持路由转发和CORS
- [ ] 统一的项目目录结构，清晰的代码组织
- [ ] 开发工具链配置 (ESLint, Prettier, pre-commit hooks)
- [ ] 环境变量配置模板和文档
- [ ] 本地PostgreSQL和Redis容器配置
- [ ] 健康检查端点 (`/health`, `/ready`)

## Technical Details
- **Docker Compose**: 配置开发环境容器网络
- **API Gateway**: 使用FastAPI创建基础路由框架
- **项目结构**: 标准化目录布局 (src/, tests/, docs/)
- **工具链**: ESLint + Prettier + Husky
- **数据库**: PostgreSQL 14 + Redis 6容器

**关键文件**:
```
docker-compose.yml
nginx.conf
gateway/main.py

.eslintrc.js
```
