# Task 1.1: 實現TimeframeManager類

## 描述
實現時間框架管理和數據重採樣功能，支持多種時間框架的回測數據處理。

## 詳細需求

### 核心功能
1. **多時間框架支持**
   - 分鐘級：1m, 5m, 15m, 30m
   - 小時級：1h, 4h, 8h
   - 日級：1d, 3d
   - 週級：1w
   - 月級：1M

2. **數據重採樣**
   - OHLCV數據聚合
   - Volume-Weighted Average Price (VWAP)
   - 內插法處理缺失數據

3. **時間框架對齊**
   - 自動檢測市場開盤時間
   - 處理不同時區的數據
   - 節假日和周末處理
