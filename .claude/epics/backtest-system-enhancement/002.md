# Task 1.2: 增強風險指標計算

## 描述
實現5種新的專業風險指標，提升回測系統的風險分析能力。

## 詳細需求

### 新增風險指標
1. **卡爾瑪比率（Calmar Ratio）**
   - 計算公式：年化收益率 / 最大回撤
   - 衡量每單位最大回撤的超額收益

2. **詹森指數（Jensen's Alpha）**
   - 基於CAPM模型計算
   - 衡量相對市場的超額收益

3. **特雷諾比率（Treynor Ratio）**
   - 計算公式：(年化收益率 - 無風險利率) / Beta值
   - 衡量每單位系統性風險的超額收益

4. **索提諾比率（Sortino Ratio）**
   - 只考慮下行波動率
   - 更準確反映投資者對虧損的厭惡

5. **最大回撤期分析**
   - 計算最大回撤持續時間
   - 回撤恢復時間統計
