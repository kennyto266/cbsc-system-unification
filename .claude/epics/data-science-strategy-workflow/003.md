# Task 003: Create visualization module (Plotly + ipywidgets)

## Description

Implement the visualization module for the CBSC Strategy SDK, providing interactive charts and controls for Jupyter notebooks. The module will use Plotly for rich visualizations and ipywidgets for interactive controls, enabling real-time data exploration and strategy analysis.

The visualization module will provide:
- Interactive OHLCV candlestick charts
- Technical indicator overlays (SMA, EMA, Bollinger Bands, RSI, MACD)
- Signal visualization (buy/sell markers)
- Performance metric charts (PnL, drawdown, returns distribution)
- Backtest results comparison
- Interactive parameter controls (sliders, dropdowns)
- Real-time data update capability

### Chart Examples

**Candlestick with Indicators:**
```python
def create_strategy_chart(
    data: pd.DataFrame,
    signals: pd.Series,
    indicators: dict[str, pd.Series]
) -> go.Figure:
    """Create complete strategy visualization."""
    builder = ChartBuilder(theme="dark")

    # Main candlestick chart
    fig = builder.candlestick(data, title="Strategy Performance")
    fig = builder.add_indicator(fig, "SMA 20", indicators["sma_20"], "blue")
    fig = builder.add_indicator(fig, "SMA 50", indicators["sma_50"], "orange")
    fig = builder.add_signals(fig, signals)

    # Add subplots for RSI and MACD
    fig = builder.subplot_indicators(fig, {
        "RSI": indicators["rsi"],
        "MACD": indicators["macd"]
    })

    return fig
```

**Performance Dashboard:**
```python
def create_performance_dashboard(
    returns: pd.Series,
    benchmark: Optional[pd.Series] = None
) -> go.Figure:
    """Create comprehensive performance dashboard."""
    charts = PerformanceCharts()

    # Create individual charts
    equity = charts.equity_curve(returns, benchmark)
    dd = charts.drawdown(returns)
    dist = charts.returns_distribution(returns)
    rolling = charts.rolling_metrics(returns)

    # Combine in layout
    manager = LayoutManager()
    dashboard = manager.dashboard(
        main_chart=equity,
        indicators=[dd, dist],
        performance=[rolling]
    )

    return dashboard
```
