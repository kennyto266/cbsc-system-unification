## Description

Create automated installation scripts and package configuration to enable easy setup of the CBSC data science strategy development environment. This includes package structure, dependency management, installation automation, and environment setup.

## Acceptance Criteria

- [ ] Package structure `setup.py` or `pyproject.toml` with proper metadata
- [ ] Installation script `scripts/install_dev.sh` (Linux/Mac) and `install_dev.bat` (Windows)
- [ ] Dependency specifications in `requirements.txt` and `requirements-dev.txt`
- [ ] Virtual environment setup in installation script
- [ ] Post-install verification script `scripts/verify_install.py`
- [ ] Configuration file templates `config/config.template.yaml`
- [ ] Environment variable setup `.env.template`
- [ ] Docker configuration for containerized development
- [ ] Installation documentation `INSTALL.md`
- [ ] Installation tested on fresh system (Linux, Mac, Windows)
- [ ] Automated tests verify installation success
- [ ] CI/CD pipeline includes installation validation

### Configuration Management

**`config/config.template.yaml`:**
```yaml
# Data sources
data:
  default_source: "yfinance"
  cache_dir: "./data/cache"
  max_cache_age_days: 7

# Backtest settings
backtest:
  initial_capital: 100000
  commission: 0.83
  slippage: 0.0001

# Indicators
indicators:
  registry: "config/indicators.yaml"

# Logging
logging:
  level: "INFO"
  file: "logs/cbsc.log"

# Jupyter
jupyter:
  notebook_dir: "notebooks"
  port: 8888
```

```bash
# API Keys
```
