# Task 84: Build ClaudeCodeAssistant with template system

## Description

Implement the `ClaudeCodeAssistant` class that integrates with Anthropic's Claude API to generate quantitative trading strategy code. The system will use a template-based approach to ensure generated code follows CBSC patterns and best practices.

The assistant will provide:
- Strategy code generation from natural language descriptions
- Parameter optimization suggestions
- Error detection and code improvement recommendations
- Template-based generation for consistent code structure
- Support for multiple strategy types (momentum, mean-reversion, arbitrage)

class StrategyTemplate(ABC):

**Template Example (Momentum):**
```python
class MomentumTemplate(StrategyTemplate):
    """Template for momentum-based strategies."""

    @property
    def strategy_type(self) -> StrategyType:
        return StrategyType.MOMENTUM

    @property
    def required_parameters(self) -> list[str]:
        return ["lookback_period", "threshold", "position_size"]

    def generate_code(
        self,
        parameters: dict,
        indicators: dict
    ) -> str:
        return f'''
class {parameters.get('name', 'MomentumStrategy')}:
    """
    Momentum Strategy: {parameters.get('description', 'N/A')}
    """

    def __init__(self):
        self.lookback = {parameters.get('lookback_period', 20)}
        self.threshold = {parameters.get('threshold', 0.02)}
        self.position_size = {parameters.get('position_size', 0.1)}

    def generate_signals(self, data: pd.DataFrame) -> pd.Series:
        """Generate trading signals."""
        # Calculate momentum
        momentum = data['close'].pct_change(self.lookback)

        # Generate signals
        signals = pd.Series(0, index=data.index)
        signals[momentum > self.threshold] = 1
        signals[momentum < -self.threshold] = -1

        return signals
'''
```
