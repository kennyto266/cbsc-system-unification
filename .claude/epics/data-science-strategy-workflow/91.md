# Task: Create Dash Dashboard with Live Updates

## Description

Build an interactive Dash dashboard that embeds directly into Jupyter notebooks and provides real-time visualization of strategy performance, market data, and backtest results. The dashboard will support live data streaming, interactive controls, and customizable layouts.

## Acceptance Criteria

- [ ] Dash application that runs in Jupyter
- [ ] Real-time chart updates from WebSocket
- [ ] Interactive controls (sliders, dropdowns, buttons)
- [ ] Multiple chart types (candlestick, line, heatmap)
- [ ] Responsive layout for different screen sizes
- [ ] Strategy performance metrics display
- [ ] Backtest results comparison view
- [ ] Export functionality (PNG, HTML)
- [ ] Dark/light theme support

### 1. Dashboard Core

Create the main Dash application:

```python
class StrategyDashboard:
    """Interactive Dash dashboard for strategy monitoring"""

    def __init__(
        self,
        workspace: 'StrategyWorkspace',
        port: int = 8050,
        mode: str = 'inline'
    ):
        self.workspace = workspace
        self.port = port
        self.mode = mode
        self.app = dash.Dash(__name__, external_stylesheets=[])
        self._setup_layout()
        self._setup_callbacks()

    def run(self):
        """Start dashboard server"""

    def _setup_layout(self):
        """Configure dashboard layout"""

    def _setup_callbacks(self):
        """Register Dash callbacks for interactivity"""
```

```python
class LiveUpdateComponent:
    """Component for real-time chart updates"""

    def __init__(self, update_interval: int = 1000):
        self.update_interval = update_interval
        self.data_buffer = CircularBuffer(size=1000)
        self.websocket = None

    def setup_websocket(self, ws_url: str, symbols: List[str]):
        """Connect to WebSocket for live data"""

    def update_layout(self):
        """Return Dash layout for this component"""

    def register_callbacks(self, app: dash.Dash):
        """Register callbacks for periodic updates"""

    @property
    def latest_data(self) -> pd.DataFrame:
        """Get latest data as DataFrame"""
```

### 6. Theme Support

Implement theming system:

```python
class ThemeManager:
    """Manage dashboard themes"""

    def __init__(self, initial_theme: str = "dark"):
        self.theme = initial_theme
        self.themes = {
            "dark": self._dark_theme,
            "light": self._light_theme
        }

    def get_stylesheet(self) -> str:
        """Get current theme stylesheet"""

    def toggle_theme(self):
        """Toggle between dark and light"""
```
