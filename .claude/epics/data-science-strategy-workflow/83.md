# Task 83: Implement StrategyWorkspace core class with data connectors

## Description

Implement the `StrategyWorkspace` class as the main entry point for the CBSC Strategy SDK. This class will provide a unified interface for data access, strategy development, and workspace management in Jupyter notebooks.

The workspace will integrate with the existing CBSC backend APIs to provide:
- Historical market data retrieval (OHLCV)
- Real-time data streaming setup
- Data caching and management
- Session state management
- Context manager interface for clean resource handling
