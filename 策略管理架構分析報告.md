## 1. 項目概述

CBSC 是一個**企業級量化交易策略回測與管理平台**，採用**微服務架構**設計。系統整合了多個服務模塊，包括用戶管理、策略執行、回測分析、實時監控等功能。

### 核心功能
- 🔐 **用戶管理系統**: 認證、授權、RBAC權限控制
- 📊 **策略管理**: 創建、修改、執行交易策略
- 📈 **回測引擎**: 基於VectorBT的高性能回測
- 📉 **實時監控**: WebSocket實時數據推送
- 🎯 **優化系統**: 參數優化和策略選擇

## 2. 技術棧

### 後端技術棧
```yaml
框架: FastAPI  (Python 3.10+)
ORM: SQLAlchemy
認證: JWT + MFA (多因子認證)
WebSocket: FastAPI WebSocket
緩存: Redis
時序數據: InfluxDB
監控: Prometheus + Grafana
日志: ELK Stack (Elasticsearch + Logstash + Kibana)
```

### 前端技術棧
```yaml
框架: React 18 + TypeScript
狀態管理: Redux Toolkit + RTK Query
UI組件:
  - Tailwind CSS
  - Ant Design
  - Radix UI
  - Headless UI
圖表庫:
  - Chart.js
  - Plotly.js
  - Recharts
構建工具: Vite
測試: Jest + React Testing Library + Playwright
路由: React Router v6
實時通信: Socket.IO Client
```

### 基礎設施
```yaml
容器化: Docker + Docker Compose
編排: Kubernetes (K8s)
數據庫: PostgreSQL 15
緩存: Redis 7
消息隊列: Redis Pub/Sub
```
